# CMSFS

This is the repository for the CMSFS (or "CMS FS") project.

## cmsfs

"cmsfs" refers to the utility capability (several related programs)
of this project.

The purpose of the utility (and of the whole projecdt) is to provide
access to VM/CMS minidisks from Linux (especially z/Linux hosted on z/VM)
or any other Unix or Unix-like environment.

"cmsfs" previously also referred to a very old Linux filesystem driver,
as in `mount -t cmsfs /device /mountpoint`. With the advent of a FUSE-based
filesystem driver for the CMS minidisk filesystem, the driver capability
in this project is not presently needed.
